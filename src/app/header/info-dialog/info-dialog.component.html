Algorithms Visualizer Guide

Quick Start Guide

  • Place a starting point my_location
  • Place an ending point place
  • Choose an pathfinding algorithm to run and click START
  • Watch the algorithm search a path from the starting to the ending point

Features

Visualization of pathfinding algorithms is very cool, so I (plan to) implement multiple algorithms such aus Breadth-First-Search, Depth-First-Search, Dijkstra's Algorithm, Swarm Algorithms, ... .
Therefore you can edit the grid by placing a STARTING POINT, an ENDING POINT and WALLS (which cannot be crossed by the searching algorithm. Furthermore a grid generator can randomly fill the grid or generate a maze.
The STARTING and ENDING POINT are also draggable, which means you can move them to see the algorithm algorithm update its path in real time.
