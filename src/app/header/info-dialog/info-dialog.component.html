Algorithms Visualizer Guide

User Guide

  • Click on empty (or highlighted / path) cells to place walls
  • Click on wall cells to remove the walls
  • Drag and drop the starting / ending point
  • Choose an pathfinding algorithm to run and click START
  • Watch the algorithm search a path from the starting to the ending point
